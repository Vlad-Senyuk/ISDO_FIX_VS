ИМПОРТ ИНФОРМАЦИИ О ДОКУМЕНТАХ ОБ ОБРАЗОВАНИИ ИЗ ФАЙЛА

 

ИСДО поддерживает импорт данных из XML-файлов и файлов Microsoft Excel. XML-файлы должны иметь следующую структуру:

<?xml version="1.0" encoding="windows-1251"?>
<diploms>
  <diplom RegNum="Номер записи в журнале о выдаче диплома">
    <Level>Уровень образования</Level>
    <DocType>Наименование документа об образовании</DocType>
   <F>Фамилия</F>
    <I>Имя</I>
    <O>Отчество</O>
    <Start>Год начала обучения</Start>
    <Stop>Год окончания обучения</Stop>
    <Create>Дата выдачи документа об образовании</Create>
    <Ser>Серия документа об образовании </Ser>
    <Num>Номер документа об образовании </Num>
    <Pasp>Личный номер гражданина</Pasp>
    <Spec>Наименование специальности</Spec>
    <Spz>Наименование специализации</Spz>
    <Kw>Наименование квалификации</Kw>
    <School>Наименование учреждение образования</School>
  </diplom>
...…
</diploms>

Наименование атрибута Значение Тип Примечание

Create Дата выдачи документа об образовании дата Значение должно соответствовать сделанной записи в документе об образовании (формат даты дд.мм.гггг)

Spz Специализация строка Значение должно соответствовать сделанной записи в документе об образовании
Kw Квалификация строка Значение должно соответствовать сделанной записи в документе об образовании

Для осуществления импорта данных из XML-файла необходимо нажать кнопку «Импорт» (см. рис. 3).
В появившемся диалоговом окне «Импорт документов об образовании» необходимо нажать кнопку «Импорт» и выбрать соответствующий файл (см. рис. 7).


Рис. 7. Окно импорта информации о документе об образовании.

При импорте данных из файла, проверка корректности сведений производится в автоматическом режиме, записи, содержащие некорректные данные, не сохраняются.
По окончанию выполнения операции импорта данных в таблице «История импорта файлов» появится запись содержащая дату импорта, наименование импортируемого файла, количество добавленных записей, количество записей, содержащих некорректную информацию и наименование файла ошибок (см. рис. 7).
Для загрузки на локальный компьютер исходного файла или файла ошибок необходимо нажать на наименование соответствующего файла (см. рис. 7).
