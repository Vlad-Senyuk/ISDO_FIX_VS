СОЗДАНИЕ ЗАПИСИ О ДОКУМЕНТЕ ОБ ОБРАЗОВАНИИ

Для добавления одной записи о документе об образовании необходимо нажать кнопку «Создать» (см. рис. 3).
В открывшемся диалоговом окне (см. рис. 5) необходимо заполнить все необходимые поля (все поля кроме поля «Специализация» являются обязательными для заполнения), после чего нажать кнопку «Сохранить».
Проверка корректности данных производится в автоматическом режиме по мере их заполнения, поле, содержащее некорректные данные, выделяется красной рамкой.
В случае успешного завершения операции сохранения данных, диалоговое окно «Документ об образовании» будет закрыто. Информация о новом документе об образовании будет добавлена в таблицу.


Рис. 5. Окно создания новой записи о документе об образовании.

Для копирования записи о документе об образовании необходимо выбрать нужную запись и нажать кнопку «Копировать» (см. рис. 3).
В открывшемся диалоговом окне (см. рис. 5) надо заполнить необходимые поля (поля из блока «Информация об образовании» и тип документа об образовании будут автоматически заполнены исходя из сведений выбранной записи), после чего нажать кнопку «Сохранить».
